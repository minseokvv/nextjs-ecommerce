# 🛒 MYOEEMYO Shop - 시스템 기능 명세서

현재 구현되어 있는 쇼핑몰 시스템의 주요 기능 목록입니다.

## 1. 👤 사용자 기능 (Customer Features)

### 1-1. 인증 (Authentication)
- **회원가입**: 아이디, 이메일, 비밀번호, 이름, 연락처, 주소를 입력하여 가입. (약관 동의 포함)
- **로그인/로그아웃**: NextAuth 기반 인증 세션 관리.

### 1-2. 쇼핑 (Shopping)
- **상품 목록**: 카테고리별 상품 조회, 검색 기능, 필터링.
- **상품 상세**: 
  - 상품 이미지 갤러리 및 상세 정보 확인.
  - 수량 선택 및 장바구니 담기 / 즉시 구매.
- **장바구니**: 상품 수량 변경, 삭제, 총 결제 금액 확인.

### 1-3. 주문 및 결제 (Order & Checkout)
- **주문서 작성**:
  - **배송지 입력**: 
    - **주소록 연동**: 저장된 배송지 중 선택 가능 (최대 5개).
    - **주소 검색**: Daum 주소 찾기 API(react-daum-postcode) 연동.
    - **연락처 자동 포맷팅**: `010-XXXX-XXXX` 형식 자동 변환.
  - **무통장 입금**: 입금자명 입력 필수, 결제 수단 선택.
- **주문 완료**: 주문 번호 발급, 입금 계좌 및 금액 안내 페이지 제공.

### 1-4. 마이페이지 (My Page)
- **주문 내역**: 전체 주문 목록 조회, 주문 상태(입금대기, 결제완료, 배송중 등) 확인.
- **주소록 관리**:
  - 배송지 추가 (주소 검색 포함), 수정, 삭제.
  - 기본 배송지 설정 기능. (최대 5개 제한)

---

## 2. 🛠 관리자 기능 (Admin Features)

### 2-1. 대시보드 (Dashboard)
- **현황판**: 일별/월별 매출 통계, 신규 주문 수, 가입자 수 등(Recharts 차트 시각화).
- **최근 활동**: 최근 들어온 주문 목록 요약 표시.

### 2-2. 상품 관리 (Product Management)
- **상품 목록**: 전체 상품 조회, 검색, 상태(판매중/품절/숨김) 필터링.
- **상품 등록/수정**:
  - 다중 이미지 업로드 (드래그 앤 드롭).
  - 카테고리, 가격, 재고, 옵션 설정.
  - 에디터를 통한 상세 설명 작성.
- **카테고리 관리**: 대분류/하위분류 카테고리 생성 및 순서 관리.

### 2-3. 주문 관리 (Order Management)
- **주문 리스트**:
  - 전체 주문 조회 및 검색 (주문번호, 주문자명, 입금자명 등).
  - **입금 확인**: 무통장 입금 주문의 '입금자명' 확인 및 결제 완료 처리.
- **상태 관리**: 주문 상태 변경 (입금대기 → 결제완료 → 배송준비 → 배송중 → 배송완료).
- **송장 등록**: 택배사 및 운송장 번호 입력.

### 2-4. 배송 설정 (Delivery Settings)
- **배송 템플릿**: 배송비, 반품/교환 안내 등 배송 관련 정책 템플릿 관리.

## 3. ⚙️ 기술 스택 (Tech Stack)

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: SQLite (Prisma ORM)
- **Authentication**: NextAuth.js
- **UI Components**: 
  - Radix UI (Dialog, Dropdown 등)
  - Lucide React (Icons)
  - Recharts (Charts)
- **External API**: Daum Postcode (주소 검색)
